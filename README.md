#Calculator Project

A clean, modern calculator built with HTML, CSS, and JavaScript featuring a sliding history sidebar.

## Features
- Basic arithmetic operations
- Slide-in / slide-out history panel
- Calculation history stored during session
- Click-friendly, responsive UI
- Clean layout and smooth animations

## How to run
Either download or clone the repo and open 'index.html' inside your browser or go through the link
**Live Link**: https://hexogreed.github.io/Calculator/

## technologies used:
-Html
-Css
-JaveScript
